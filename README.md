# 项目目录结构说明

| 文件夹               | 说明                                                         
|---------------------|--------------------------------------------------------------
| /Configs            | 配置相关，如协议参数、常量定义等。                            
| /Enums              | 枚举类型定义，如协议类型、数据类型等。                        
| /Extensions         | 扩展方法，如最小API的 Map 扩展、服务注册扩展等。               
| /Helpers            | 工具类和辅助方法，如参数校验、格式转换等。                    
| /Middlewares        | 中间件，如全局异常处理、日志记录等。                          
| /Models             | 数据模型和DTO，包括 Protocol、Device、Point 等业务实体。       
| /Protocols          | 协议适配器层，每种工业协议实现一个 Adapter，统一协议接口。 
| /Protocols/Inteface  | 接口相关，协议工厂接口和协议接口。
| /Protocols/Factory  | 协议工厂相关，负责根据协议类型动态创建协议适配器实例。         
| /Services           | 业务服务层，负责协议调度、设备管理、读写操作等核心业务逻辑。
